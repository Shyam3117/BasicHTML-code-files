CSS (Cascading Style Sheets)
----------------------------
	-> Which is for creating a responsive page with better UI(User Interface)
	-> By using style tag we are going to create the css logic 
Syntax:
	selector{
		property: value;
		property: value;
		property: value;
		.
		.
		.	
		.	
		.	
		.
	}

	-> selector : It means selecting a html element which happens using the ways represented below:
			1.simple selector:
				element(img, h1, p, pre,....), id(#), class(.)
			2. combinator selector:
				descandant( ), child(>), adjascant sibiling(+), general sibiling(~)
			3. pseudo class:
				:link, :visited, :hover, :active
			4. pseudo element:
				::first-line, ::last-line, ::first-letter, ::last-letter, ::before, ::after
			5. attridute:
				based on element attribute the styling will be applied
